Using

“Ok, so how do i actually get an application/website to use this font”

unfortunately, there isn't a simple answer to that.
most things will either use whatever the default emoji font is, or they'll have a list of fonts that they check for.

basically, the process is different for each application and website and operating system.

note: i will talk about the "name" of a font. this is determined by metadata in the font, not by the filename. (i.e. renaming the file won't do anything, you have to use a tool to change the strings in the name table)

Firefox

in about:config, change font.name-list.emoji to Twemoji COLR. This will set the emoji font used in the UI and on any website which doesn't specify an emoji font.

Websites

many websites will specify a list of emoji fonts beginning with "Apple Color Emoji", "Segoe UI Emoji", ... (the first one you have installed will be used).

so, installing twemoji with the name "Apple Color Emoji" is actually a fairly reliable option (unless you are on MacOS, of course). This works on many websites such as Bluesky, Github, etc.

Windows/MacOS

you may be able to override the builtin emoji font by installing twemoji as the same name ("Segoe UI Emoji" on windows, and "Apple Color Emoji" on mac) however, this can cause problems because applications may expect the emoji font to have certain features (e.g. black and white glyphs for programs which can't display color glyphs)
